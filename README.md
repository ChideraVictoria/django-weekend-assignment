# django-weekend-assignment - TechRise
## Overview
This project contains three apps:ToDo,Quotes,and Contact.

## How to Run
Clone the repo
Create a virtual environment
Run 'pip install -r requirements.txt'
Run 'python manage.py runserver'

##  Features
-Task manager(CRUD)
-Motivational quotes
-Contact page
